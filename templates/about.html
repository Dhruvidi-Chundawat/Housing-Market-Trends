ℹ️ About This Project

Housing Market Trends Analysis

📋 Project Overview

This project provides a comprehensive analysis of housing market trends using data visualization and interactive analytics. The dataset contains 21,609 housing transaction records with detailed information about property characteristics, pricing, and renovation status.

🎯 Project Goals

  • Provide clear insights into housing market patterns and trends
  • Enable data-driven decision making for real estate stakeholders
  • Visualize relationships between property age, features, and pricing
  • Understand renovation patterns and their impact on market values
  • Create an interactive platform for market exploration

📊 Data Specifications

Dataset Name: Transformed_Housing_Data2

Total Records: 21,609 housing transactions

Key Metrics:

  • Sale Price (Range: $75,000 - $1,129,575)
  • Property Age (Range: 3 - 118 years)
  • Number of Bedrooms, Bathrooms, and Floors
  • Basement and Living Areas
  • Renovation Status and Years Since Renovation
  • Zipcode Groups for Geographic Analysis

🔧 Technology Stack

  • Data Visualization: Tableau Public
  • Backend: Flask (Python)
  • Frontend: HTML5, CSS3
  • Data Processing: Python (Pandas, NumPy)

📈 Key Visualizations

  • KPI Cards: Quick overview of critical metrics
  • Histogram: Sales distribution by renovation timeline
  • Pie Chart: House age distribution by renovation status
  • Grouped Bar Chart: Property features analysis by age group
  • Interactive Filters: Custom analysis by price and age

🎓 Key Insights

  • Renovation Rate: Only 4.2% of properties have been renovated (914 out of 21,609)
  • Average Price: The median sale price is $450,000 with a mean of $511,619
  • Age Distribution: Properties range from 3 to 118 years old, with most clustering in 20-50 year range
  • Feature Correlation: Newer homes tend to have more bathrooms and bedrooms
  • Market Segments: Both heritage and modern properties represent significant market segments

💼 Use Cases

  • Real Estate Agents: Understand market trends and pricing strategies
  • Investors: Identify market opportunities and property value patterns
  • Researchers: Analyze housing market dynamics and trends
  • Policy Makers: Understand housing stock composition and renovation needs
  • Developers: Identify gaps in the market and renovation potential

📞 Technical Support

This dashboard is built using Flask and Tableau Public. For questions about the data or analysis, please refer to the documentation or contact the project team.
